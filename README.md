# 术后滴眼药水管理

一个纯静态网页，用来管理半飞秒术后滴眼药水计划。

## 功能

- 根据药水每天次数自动生成时间轴
- 多种药水自动按间隔排队
- 支持按推荐时间提醒，晚点后仍按顺序补最早未完成的一次
- 支持甘特图式今日视图，快速看每瓶药的全天分布
- 支持手动标记已滴、撤销、延后
- 支持在网页里维护药水数据和用药周期
- 支持导出/导入配置
- 可部署到 GitHub Pages

## 使用

直接用浏览器打开 `index.html` 即可。

数据保存在浏览器 `localStorage` 中。不同设备访问同一个 GitHub Pages 地址时，页面代码相同，但本地记录不会自动同步。需要迁移时可以在“设置”里导出配置，再到另一台设备导入。

## 部署到 GitHub Pages

1. 创建 GitHub 仓库。
2. 把本目录提交并推送到仓库。
3. 在仓库 `Settings > Pages` 中选择 `Deploy from a branch`。
4. Branch 选择 `main`，目录选择 `/root`。

部署后即可通过 GitHub Pages 的公网地址访问。

## 医嘱提醒

默认药水配置是按图片和常见术后表格整理的起点，实际用药名称、次数、天数和递减规则应以医生给你的处方为准。
